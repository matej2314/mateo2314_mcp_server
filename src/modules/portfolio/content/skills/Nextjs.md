React pod routing, SSR/SSG i produkcję rozwijam w Next.js — **framework** z renderowaniem po stronie serwera i optymalizacjami od stron marketingowych po aplikacje z API routes / server actions. Potrafię **budować layouty i segmenty routingu** (App Router), wybierać **strategię renderowania** pod SEO i UX (SSR, SSG, dynamiczne segmenty) oraz integrować **middleware i konfigurację obrazów**.

Znam **podstawy modelu żądań i cache w Next** oraz stosuję **praktyki utrzymaniowe**: jasny podział komponentów server/client, świadome użycie `fetch` i rewalidacji, typowanie z TypeScript oraz świadome ograniczenia — złożony stan klienta czasem lepiej izolować, a koszty cold start i zależności od hostingu warto planować wcześnie.

Doświadczenie z:
- aplikacjami pełnostackowymi i witrynami z treścią SEO
- integracją z bazami i zewnętrznymi API po stronie serwera
- wdrożeniami na platformach wspierających Node (Vercel, Docker, inne)
